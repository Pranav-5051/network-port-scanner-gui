# 🌐 Network Port Scanner GUI

## 📌 Description
A Python-based Network Port Scanner with a user-friendly GUI built using Tkinter.  
This application allows users to scan a target IP address or hostname and detect open ports along with their associated services.

The tool uses socket programming and multi-threading to perform fast and efficient scanning, making it useful for cybersecurity learning and network analysis.

---

## 🚀 Features
- ⚡ Multi-threaded fast scanning (500 threads)
- 📊 Real-time progress tracking
- 🔍 Detects open ports and services (FTP, HTTP, etc.)
- 🖥️ Simple and clean GUI interface
- ⛔ Stop scan anytime
- 💾 Save results to file
- 🌍 Cross-platform support (Windows/Linux/macOS)

---

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- Socket Programming
- Threading & Queue

---

## ▶️ How to Run

1. Install Python (if not installed)  
2. Download or clone the repository  
3. Run the following command:

```bash
python portscanergui.py
```
